#Append Arrays</h1>

Write a program to append several arrays of numbers.
-	Arrays are separated by ‘|’.
-	Values are separated by spaces (‘ ’, one or several).
-	Order the arrays from the last to the first, and their values from left to right.

##Examples

>Input

    1 2 3 |4 5 6 |  7  8
>Output

    7 8 4 5 6 1 2 3
